## 🏗️ 技术实现

### 后端技术栈 ✅
- **Node.js** - v14+ 运行环境
- **Express** - Web 框架
- **SQLite** - 轻量级数据库
- **bcrypt** - 密码加密
- **jsonwebtoken** - JWT 认证
- **nodemailer** - 邮件发送
- **node-cron** - 定时任务
- **cors** - 跨域支持
- **body-parser** - 请求解析

### 前端技术栈 ✅
- **HTML5** - 语义化结构
- **CSS3** - 现代样式（Grid、Flexbox、渐变、动画）
- **JavaScript ES6+** - 原生 JS，无框架依赖
- **Canvas API** - 图表绘制
- **Fetch API** - HTTP 请求
- **LocalStorage** - 本地存储

### 数据库设计 ✅
- **users** 表 - 用户信息
- **checkins** 表 - 签到记录
- **contacts** 表 - 联系人信息
- **notifications** 表 - 通知记录

## 🚀 部署说明

### 本地运行
```bash

npm install
```

## 📝 API 接口

### 认证接口
- `POST /api/auth/register` - 用户注册
- `POST /api/auth/login` - 用户登录

### 签到接口
- `POST /api/checkin` - 每日签到
- `GET /api/checkin/stats` - 签到统计

### 联系人接口
- `POST /api/contacts` - 添加联系人
- `GET /api/contacts` - 获取联系人列表
- `DELETE /api/contacts/:id` - 删除联系人
- `PUT /api/contacts/:id` - 更新联系人

### 通知接口
- `GET /api/notifications` - 获取通知历史
