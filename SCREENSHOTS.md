## 设计系统

### 色彩
- **主色**：#667eea → #764ba2（渐变紫）
- **辅助色**：#f093fb → #f5576c（渐变粉）
- **背景**：#ffffff（白色）
- **文字**：#333333（深灰）
- **边框**：#e0e0e0（浅灰）

### 字体
- 系统默认字体栈
- 中文：PingFang SC, Microsoft YaHei
- 英文：-apple-system, Segoe UI

### 圆角
- 卡片：20px
- 按钮：10px
- 输入框：10px
- 小元素：8px

### 阴影
- 卡片：0 10px 30px rgba(0, 0, 0, 0.2)
- 按钮悬停：0 10px 30px rgba(102, 126, 234, 0.4)
- 输入框聚焦：0 0 0 3px rgba(102, 126, 234, 0.1)

### 动画
- 呼吸动画：3s ease-in-out infinite
- 淡入淡出：0.5s ease-out
- 按钮悬停：0.3s ease
- 过渡效果：all 0.3s ease

## 响应式设计

### 桌面端（> 768px）
- 最大宽度 800px
- 居中显示
- 双列布局（状态选择器）

### 平板端（768px - 480px）
- 自适应宽度
- 导航栏垂直排列
- 单列布局

### 移动端（< 480px）
- 全宽布局
- 更小的字体和间距
- 触摸友好的按钮尺寸

## 交互细节

### 按钮
- 悬停：向上移动 2px + 阴影增强
- 点击：无边框，渐变背景
- 禁用：灰色背景，不可点击

### 输入框
- 聚焦：边框变色 + 外发光
- 错误：红色边框 + 错误提示
- 成功：绿色边框

### Toast 提示
- 右上角弹出
- 3 秒后自动消失
- 成功：绿色背景
- 错误：红色背景

### 卡片
- 白色背景
- 圆角阴影
- 淡入动画
- 充足的内边距
