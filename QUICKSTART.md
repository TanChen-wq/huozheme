# 快速开始指南

## 🚀 三步启动

### 1️⃣ 安装依赖
```bash
npm install
```

### 2️⃣ 启动服务器
```bash
npm start
```

### 3️⃣ 打开浏览器
访问：**http://localhost:3000**

---

## 📱 完整使用流程

### 步骤 1：注册账号
1. 打开首页
2. 点击"立即注册"
3. 填写信息：
   - 用户名：`测试用户`
   - 邮箱：`test@example.com`
   - 密码：`123456`
4. 点击"注册"

### 步骤 2：登录
1. 使用刚才注册的邮箱和密码登录
2. 自动跳转到个人面板

### 步骤 3：添加联系人
1. 点击导航栏的"联系人"
2. 填写联系人信息：
   - 姓名：`家人`
   - 邮箱：`family@example.com`
3. 点击"添加联系人"

### 步骤 4：每日签到
1. 点击导航栏的"签到"
2. 选择今天的状态（例如：😊 很好）
3. 可选：输入留言"今天心情不错"
4. 点击"签到"
5. 查看控制台，会看到通知发送日志

### 步骤 5：查看统计
1. 点击导航栏的"统计"
2. 查看连续签到天数和累计签到次数
3. 查看最近 7 天的签到趋势图

## 📝 注意事项

### 演示模式
默认情况下，邮件和短信通知只会在控制台打印，不会真实发送。

查看通知日志：
```bash
# 在服务器运行的终端中查看
=== 邮件通知 ===
收件人: family@example.com
主题: 测试用户 今日签到通知
内容: ...
================
```

### 启用真实邮件
如需启用真实邮件发送，请参考 `STARTUP.md` 中的"邮件配置"部分。

### 数据持久化
所有数据存储在 `data.db` 文件中，删除该文件即可重置数据。

## 🔧 常用命令

```bash
# 启动服务器
npm start

# 开发模式（需要安装 nodemon）
npm run dev

# 停止服务器
# 在终端按 Ctrl+C

# 重置数据库
rm data.db && npm start

# 查看项目结构
tree -L 2 -I 'node_modules'

# 打包项目
tar --exclude='node_modules' --exclude='data.db' -czf huozheme.tar.gz huozheme
```

## 💡 快速测试场景

### 场景 1：测试签到功能
```
1. 注册并登录
2. 添加一个联系人
3. 进行签到
4. 查看控制台的通知日志
5. 刷新页面，验证今日已签到
```

### 场景 2：测试连续签到
```
1. 第一天签到
2. 修改系统时间到第二天（或等待）
3. 第二天签到
4. 查看统计页面，连续签到天数应为 2
```

### 场景 3：测试未签到提醒
```
1. 注册并添加联系人
2. 签到一次
3. 等待超过 24 小时（或修改数据库中的 last_checkin）
4. 等待定时任务执行（每小时整点）
5. 查看控制台的提醒日志
```

### 场景 4：测试多用户
```
1. 注册用户 A
2. 退出登录
3. 注册用户 B
4. 两个用户分别签到
5. 验证数据独立性
```

## 🆘 遇到问题？

### 问题 1：端口被占用
```bash
# 修改端口
PORT=3001 npm start
```

### 问题 2：依赖安装失败
```bash
# 清理缓存
npm cache clean --force
rm -rf node_modules package-lock.json
npm install
```

### 问题 3：数据库错误
```bash
# 重置数据库
rm data.db
npm start
```

### 问题 4：页面无法访问
```bash
# 检查服务器是否启动
# 应该看到：服务器运行在 http://localhost:3000

# 检查防火墙
# 确保 3000 端口未被阻止
```
